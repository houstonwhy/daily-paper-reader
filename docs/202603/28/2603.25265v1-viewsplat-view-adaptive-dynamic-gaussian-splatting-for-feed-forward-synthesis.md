## 摘要
我们提出了 ViewSplat，这是一种视角自适应的 3D 高斯泼溅网络，用于从无位姿图像中进行新视角合成。虽然最近的前馈 3D 高斯泼溅通过绕过逐场景优化显著加速了 3D 场景重建，但仍存在根本性的保真度差距。我们将这一瓶颈归因于单步前馈网络在回归满足所有视角的静态高斯原语方面的能力有限。为了解决这一局限性，我们将范式从静态原语回归转向视角自适应的动态泼溅。我们的流水线学习的是一种视角可自适应的潜表示，而非僵化的高斯表示。具体而言，ViewSplat 最初预测基础高斯原语以及动态 MLP 的权重。在渲染过程中，这些 MLP 以目标视角坐标作为输入，并预测每个高斯属性（即 3D 位置、缩放、旋转、不透明度和颜色）的视角相关残差更新。我们将这种机制称为视角自适应动态泼溅，它允许每个原语纠正初始估计误差，从而有效地捕捉高保真外观。大量实验表明，ViewSplat 在保持快速推理（17 FPS）和实时渲染（154 FPS）的同时，实现了最先进的保真度。

## Abstract
We present ViewSplat, a view-adaptive 3D Gaussian splatting network for novel view synthesis from unposed images. While recent feed-forward 3D Gaussian splatting has significantly accelerated 3D scene reconstruction by bypassing per-scene optimization, a fundamental fidelity gap remains. We attribute this bottleneck to the limited capacity of single-step feed-forward networks to regress static Gaussian primitives that satisfy all viewpoints. To address this limitation, we shift the paradigm from static primitive regression to view-adaptive dynamic splatting. Instead of a rigid Gaussian representation, our pipeline learns a view-adaptable latent representation. Specifically, ViewSplat initially predicts base Gaussian primitives alongside the weights of dynamic MLPs. During rendering, these MLPs take target view coordinates as input and predict view-dependent residual updates for each Gaussian attribute (i.e., 3D position, scale, rotation, opacity, and color). This mechanism, which we term view-adaptive dynamic splatting, allows each primitive to rectify initial estimation errors, effectively capturing high-fidelity appearances. Extensive experiments demonstrate that ViewSplat achieves state-of-the-art fidelity while maintaining fast inference (17 FPS) and real-time rendering (154 FPS).

*   **研究背景**：3D 高斯泼溅（3DGS）在实时高保真渲染方面表现出色。为了避免逐场景优化的耗时，前馈式（Feed-forward）3DGS 网络应运而生，能够直接从少量图像重建场景。
*   **核心问题**：现有的前馈式方法在保真度上与基于优化的方法存在显著差距。作者认为瓶颈在于：单步前馈网络试图回归“静态”的高斯原语，这难以同时满足所有视角的观测需求，尤其是在处理复杂几何和非朗伯体（如高光、反射）表面时。
*   **研究动机**：打破静态原语回归的范式，引入一种能够根据目标视角动态调整高斯属性的机制，从而在不牺牲速度的前提下提升重建精度。

### 2. 方法论
*   **核心思想**：提出“视角自适应动态泼溅”（View-Adaptive Dynamic Splatting）。网络不再预测一组固定的高斯参数，而是预测基础高斯原语以及一个动态 MLP（由超网络生成）的权重。
*   **关键技术细节**：
    1.  **架构组成**：基于几何 Transformer（如 MASt3R 或 VGGT）提取特征，包含位姿估计头、高斯中心头、高斯参数头以及新增的**视角相关头（View-dependent Head）**。
    2.  **超网络机制**：视角相关头根据输入图像特征，为每个像素动态生成一个小型“视角 MLP”的参数。
    3.  **视角自适应更新**：在渲染阶段，将目标视角的坐标（重参数化为 3D 单位方向和 1D 对数距离的 4D 特征）输入视角 MLP，预测高斯属性（位置 $\mu$、不透明度 $\alpha$、旋转 $r$、缩放 $s$、颜色/谐波系数 $c$）的**残差偏移量（Residual Updates）**。
    4.  **渲染流程**：基础属性 + 残差偏移 = 最终高斯属性 $\rightarrow$ 传统 3DGS 光栅化渲染。
*   **公式/流程简述**：$\hat{G} = G_{base} + \Delta G(Pose_{target}, \theta_{dynamic})$，其中 $\theta_{dynamic}$ 是由场景上下文决定的动态权重。

### 5. 实验数量与充分性
*   **实验规模**：在两个大规模数据集（RE10K, ACID）上进行了详尽的定量对比，并在 DTU 上进行了跨域泛化测试。
*   **消融实验**：针对残差组件（位置、不透明度等）、训练策略（初始化方式、是否冻结）、视角重参数化方式、MLP 隐藏层维度等进行了多组实验。
*   **充分性评价**：实验设计客观且公平，对比了不同重叠度（小、中、大）下的表现，涵盖了从室内到室外、从场景到物体的多种情况，验证了该方法在不同骨干网络（SPFSplat 各变体）上的普适性。

### 6. 主要结论与发现
*   **性能领先**：ViewSplat 在所有基准测试中均达到了最先进（SOTA）的水平，尤其在捕捉高频细节和非朗伯体效应方面表现优异。
*   **动态修正的必要性**：实验证明，同时更新位置和不透明度对保持结构一致性至关重要；仅增加球谐函数（SH）阶数无法解决前馈模型的保真度瓶颈，而动态残差更新效果显著。
*   **泛化能力**：即使在 RE10K 上训练，模型也能很好地泛化到未见过的自然场景和物体。

### 7. 优点
*   **解决瓶颈**：通过动态更新机制，有效弥补了前馈模型与优化模型之间的保真度差距。
*   **无需位姿**：完全自监督，不需要预计算相机位姿（SfM），极大地扩展了应用场景。
*   **高效平衡**：在显著提升画质的同时，依然保持了前馈网络的高速推理和 3DGS 的实时渲染特性。
*   **即插即用**：视角自适应模块可以作为通用组件集成到多种前馈 3DGS 架构中。

### 8. 不足与局限
*   **未观测区域限制**：由于是基于重建而非生成模型，ViewSplat 无法“幻觉”出完全未观测到的区域，在稀疏视角下仍会出现模糊或空洞。
*   **渲染开销**：相比静态 3DGS，由于每个视角都需要通过 MLP 计算残差，渲染速度虽仍达实时，但较原始 3DGS 有所下降。
*   **依赖骨干网络**：其性能上限在一定程度上受限于底层几何 Transformer 提取特征的质量。
